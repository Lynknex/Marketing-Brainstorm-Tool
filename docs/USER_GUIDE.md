# User Guide

## Getting Started

### Creating Your First Campaign

1. **Sign In**
   - Navigate to the application
   - Sign in with Google or email/password

2. **Create Campaign**
   - Click "New Campaign" on the dashboard
   - Enter campaign name (e.g., "April 2026 LinkedIn Campaign")
   - Click "Create Campaign"

3. **Brainstorm Flow**
   - Answer 6 guided questions:
     1. Campaign objectives (lead gen, brand awareness, thought leadership)
     2. Target audience (decision makers, technical, end users)
     3. Posting frequency (conservative, moderate, aggressive)
     4. Content mix (image-heavy, video-first, balanced, text)
     5. Product focus (cameras, tablets, ElevateX, NeuraNet)
     6. Success metrics (engagement, leads, traffic, followers)

4. **AI Generation**
   - Claude generates complete campaign design
   - Review generated posts
   - Export design specification

### Managing Campaigns

**Campaign Dashboard**
- View all campaigns with status badges
- See post counts and date ranges
- Quick access to calendar view

**Calendar View**
- Week-by-week visualization
- Click posts to see full details
- Navigate between weeks with arrow buttons

**Post Details Modal**
- Full copy text
- Visual concept description
- Hashtags and CTA
- Product focus and holiday tie-ins

### Exporting Campaigns

**Markdown Export**
- Click "Export Markdown" button
- Downloads `.md` file with full campaign
- Perfect for version control
- Copy/paste into other tools

**PDF Export**
- Click "Export PDF" button
- Downloads formatted `.pdf` document
- Share with stakeholders
- Print for offline review

## Best Practices

### Campaign Planning

1. **Start Early** - Begin 2-4 weeks before launch date
2. **Be Specific** - Clear objectives produce better AI results
3. **Know Your Audience** - Select precise personas
4. **Mix Content** - Balance text, images, and video

### Using AI Suggestions

- Review all AI-generated content before publishing
- Customize copy to match your brand voice
- Adapt visual concepts to available assets
- Adjust posting schedule based on your availability

### Content Production Workflow

1. Export campaign design (markdown or PDF)
2. Create content in batches (all text posts, then images, then videos)
3. Get stakeholder approvals
4. Schedule posts in LinkedIn
5. Monitor engagement daily

## Troubleshooting

**Q: AI generation is slow**
A: Campaign generation takes 30-90 seconds. Be patient. Claude is creating 10-20 detailed post concepts.

**Q: Export button doesn't work**
A: Check browser pop-up blocker. The export opens in a new tab.

**Q: Can't see my campaigns**
A: Verify you're logged in with the correct account.

**Q: Need to change campaign dates**
A: Currently, campaign dates are set at creation. Future updates will allow editing.

## Tips & Tricks

- **Templates:** Save successful post formats for reuse
- **Tracking:** Note which content types perform best
- **Iteration:** Adjust strategy based on monthly analytics
- **Batching:** Produce all content at once for efficiency
- **Backup:** Export campaigns regularly as markdown files
